自有短连接
/qsub?url=机场订阅   qxlist订阅
/csub?url=机场订阅   clashlist订阅
/qrule?url=Base64后的规则链接(surge规则为主)   qx规则
/crule?url=Base64后的规则链接(surge规则为主)    clash规则
/iprule?url=Base64后的规则链接(surge规则为主)   ipcidr规则
